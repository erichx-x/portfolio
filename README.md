
# Erich Casagrande Perusso - Portfolio

## Overview

This portfolio project was created to showcase my skills as a Front-End Developer and UX/UI Designer. The portfolio includes examples of my design and development work, highlighting my creative process from prototyping to final implementation.

## Technologies Used

- **Design**: Figma
- **Development**: HTML, CSS, JavaScript
- **Version Control**: Git & Bitbucket

## Project Structure

1. **Prototypes**
   - I used Figma to create detailed prototypes, focusing on usability and user experience.
   - Includes wireframes, user flows, and high-fidelity mockups.

2. **Layout Implementation**
   - Converted Figma prototypes into code using HTML for structure, CSS for styling, and JavaScript for interactivity.
   - Added animations and transitions to enhance user experience.

3. **Source Code**
   - All source code is available in this GitHub repository.
   - Includes detailed comments and documentation for easy understanding and maintenance.

## Features

- **Responsiveness**
  - The layout is fully responsive, ensuring a great experience on mobile devices, tablets, and desktops.
  
- **Interactivity**
  - Added interactive elements such as sliding menus, modals, and hover animations.

- **Accessibility**
  - Followed best accessibility practices to ensure the portfolio is usable by people with disabilities.

## How to Run

1. Clone the repository:
   ``
   git clone https://github.com/ErichCasagrandePerusso/portfolio.git
   ``

2. Navigate to the project directory:
   ``
   cd portfolio
   ``

3. Open the \`index.html\` file in your preferred browser.
